# Хоккейный Состав - PWA приложение

Веб-приложение для формирования состава на хоккейную игру с поддержкой PWA (Progressive Web App).

## Возможности

- ✅ Управление командой (добавление/удаление игроков)
- ✅ Формирование состава из 15 игроков
- ✅ Drag & Drop для добавления игроков в состав
- ✅ Dropdown меню для выбора игроков
- ✅ Сохранение данных в localStorage
- ✅ PWA поддержка (офлайн работа, установка на устройство)
- ✅ Адаптивный дизайн

## Установка и запуск

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

## Использование

1. **Добавление игроков**: Нажмите кнопку "+ Добавить игрока" и заполните форму
2. **Добавление в состав через Drag & Drop**: Перетащите игрока из списка команды на свободную позицию в составе
3. **Добавление через Dropdown**: Выберите игрока из выпадающего списка на свободной позиции
4. **Удаление из состава**: Нажмите кнопку "Удалить" на позиции в составе

## PWA иконки

Для полной поддержки PWA необходимо добавить иконки:
- `icon-192.png` (192x192 пикселей)
- `icon-512.png` (512x512 пикселей)

Вы можете создать их с помощью любого графического редактора или онлайн-генератора иконок.
